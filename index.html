Panduan Singkat Sinkronisasi

  1. Buat Google Spreadsheet baru di Drive Anda.
  2. Buka menu Ekstensi > Apps Script, tempelkan kode Apps Script Anda.
  3. Klik Deploy > Penerapan Baru, pilih Web App (Aplikasi Web).
  4. Atur Akses sebagai "Siapa Saja (Anyone)" lalu terapkan.
  5. Salin tautan URL Web App tersebut ke input formulir di samping kiri.
  6. Klik tombol Hubungkan & Sinkronkan.
